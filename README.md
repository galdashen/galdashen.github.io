# galdashen.github.io

个人学习笔记站点，基于 [Docusaurus](https://docusaurus.io/) 构建，部署在 [GitHub Pages](https://galdashen.github.io)。

## 🚀 快速开始

**安装依赖**

```bash
npm install
```

**启动本地开发服务器**

```bash
npm start
```

**构建生产版本**

```bash
npm run build
```

**部署到 GitHub Pages**

可用 GitHub Actions 自动构建并部署。

如需手动部署：

```bash
npm run deploy
```
